## Geo Instagram

Explore places and corresponding images/videos by location

### Features

- [Yes] Map View of interesting nearby locations
- [Yes] Detailed View of images/videos clicked with selected location tag
- [Yes] Add new locations based on the current map position
- [Yes] Add new pins only when map is dragged enough
- [Yes] No new pins added when the map is zoomed out
